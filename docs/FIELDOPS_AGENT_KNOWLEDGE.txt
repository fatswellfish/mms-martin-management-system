# fieldops_AGENT_KNOWLEDGE.txt

## 1. Agent 范围与能力 (Agent Scope & Capabilities)
- **可操作范围**: 严格限于 `mms/fieldops/` 及其子目录（`event_engine/`）。
- **可执行任务**:
  - 生成并修改 `models.py`, `schemas.py`, `service.py`, `router.py` 等核心代码文件。
  - 创建和填充 `event_engine/` 子模块的文件结构。
  - 生成技术文档和调试脚本。
  - 执行结构迁移任务，但仅限于 `fieldops/` 内部。
- **不可操作**: 严禁访问或修改 `mms_legacy/` 或 `system_core/` 目录下的任何文件。
